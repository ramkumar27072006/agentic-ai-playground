# Agentic AI Playground

A contributor-first open-source hub for AI agents, workflows, evaluations, benchmarks, and experiments.

## Contribute

We actively welcome:
- Bug reports
- Feature requests
- Research ideas
- Benchmark contributions
- Pull requests

## Good First Contributions

- Add agent examples
- Improve documentation
- Add benchmark datasets
- Create tutorials
- Improve evaluation pipelines

## Vision

Build a community-driven repository where contributors can learn, experiment, and publish practical agentic AI implementations.
