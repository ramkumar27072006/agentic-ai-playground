# Agent Benchmarks

## Mission

Develop open and reproducible benchmarks for evaluating AI agents.

## Evaluation Categories

### Tool Use

Measure effectiveness of external tool interactions.

### Planning

Evaluate multi-step reasoning and execution.

### Reliability

Measure consistency across repeated runs.

### Safety

Evaluate robustness against misuse and prompt injection.

### Efficiency

Track latency, token usage, and execution cost.
