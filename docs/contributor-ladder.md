# Contributor Ladder

Agentic AI Playground rewards consistent contribution and learning.

## Level 1 — Explorer

- First issue opened
- First discussion started
- First pull request submitted

## Level 2 — Builder

- 3 merged pull requests
- Documentation or examples contribution

## Level 3 — Research Contributor

- Benchmark contribution
- Research proposal accepted
- Experimental implementation merged

## Level 4 — Core Contributor

- Significant project ownership
- Mentoring contributors
- Reviewing pull requests

## Level 5 — Maintainer Candidate

- Sustained impact
- Community leadership
- Technical excellence
