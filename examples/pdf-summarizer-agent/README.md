# PDF Summarizer Agent

## Overview

A simple Agentic AI example that extracts text from PDF files and generates concise summaries.

## Features

- PDF text extraction
- AI-powered summarization
- Minimal setup
- Beginner-friendly example

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python main.py sample.pdf
```

- OCR support
